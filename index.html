Commands Usage
ctrl+s saving the fail
ctrl+c copying the text
ctrl+v paste the text
ctrl+x cut the text
ctrl+a select the code
ctrl+z for undo
ctrl+y for redo
ctrl+ zoom in
ctrl- zoom out
shift+up for shift selecting the upside of the text
shift+down for shift selecting the downside of the text
shift+right arrow selecting the right side of the text
shift+left arrow selecting the left side of the text
alt+up it moves the line one level up
alt+down it moves the line one level down
shift+alt+down arrow copy line down
shift+alt+up arrow copy line up
ctrl+shift+alt+down writing the code multiple lines
